struct  rbf_sounder_param_t
 Sounder control parameters Sounder mode represents a persistent state, while sounder action represents an action. You can use the sounder mode parameter alone to put the sounder into a certain alarm mode, while setting the sounder action to RBF_SOUNDER_CTRL_DISABLED. Alternatively, you can use the sounder action alone to instruct the sounder to perform a specific action, while setting the sounder mode to RBF_SOUNDER_MODE_DISABLED. When you set both the sounder mode and sounder action, the sounder will first execute the action and then enter the sounder mode. More...
