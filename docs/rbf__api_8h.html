int rbf_device_led_indicate_set ( RBF_dev_id_t *  id,
RBF_led_indicate_t *  indicate 
)

Set LED indicator light status for RBF sub-devices.

Parameters
idSub-device addressing address
indicateLED indicator light status
Returns
int 0: Set successful, -1: Set failed
Note
For battery-powered devices, the LED indicator light status needs to be sent in the transmission window when the sub-device reports information. Use cases include: Remote control LED indicator light for arming/disarming status; when a user presses the arming/disarming button on the remote control, the application receives the key value and sends the LED light indicator status. The remote control receives the previous LED light indicator status and provides the corresponding indication.

◆ rbf_start_findme()

int rbf_start_findme ( RBF_dev_id_t *  ids,
unsigned char  count 
)

Enable Find Me broadcast.

Parameters
idsList of RBF devices to enable Find Me
countNumber of devices in the list to enable Find Me
Returns
int 0: Successfully enable Find Me mode, -1: Failed to enable Find Me mode
Note
Find Me broadcast is a bidirectional broadcast. If a device does not respond after the broadcast, a retransmission will be initiated based on the retry count until a response is successfully received
Example:
RBF_dev_id_t ids[2];
ids[0].type = RBF_DEV_IO;
ids[0].no = 1;
ids[1].type = RBF_DEV_IO;
ids[1].no = 2;
rbf_start_findme(ids, 2);
